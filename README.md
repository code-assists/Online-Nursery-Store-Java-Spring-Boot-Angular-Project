# 🌱 Online Nursery Store | Java Spring Boot, Angular & MySQL Project  

**Online Nursery Store** is a **full-stack web application** developed using **Java Spring Boot, Angular, and MySQL**.  
This project manages **Plant Items, Categories, Companies, Customers, Orders, and Sales**.  

The application supports **two user roles**:  
- **Admin** → Manage plant items, categories, customers, orders, and reports.  
- **Customer** → Register, log in, browse plant items, add to cart, place orders, make payments, and provide feedback.  

This is a **major final year project** with **source code, MySQL database, and documentation** included for students.

## 📖 Abstract  

The **Online Nursery Store Project** simplifies buying and selling plants online.  
- **Admins** → Can add, edit, delete, and manage plant items, categories, and customer orders.  
- **Customers** → Can browse plants, order them, give feedback, and track their orders.  

It’s an excellent **Angular + Spring Boot full-stack project** for **Computer Science students** working on final year assignments.  

---

## 📂 Modules & Features  

### 👤 Customer Module  
- Admin can add, edit, update, and delete customers.  
- Customers can view & update their own details.  
- CRUD operations with restricted access for customers.  

### 🌿 Plant Item Module  
- Admin can manage plant items (CRUD).  
- Customers can view plant item details.  

### 🗂 Plant Item Category Module  
- Admin can manage plant categories (CRUD).  
- Customers can view categories.  

### 🛒 Order Module  
- Admin manages customer orders (CRUD).  
- Customers can view their orders & receipts.
